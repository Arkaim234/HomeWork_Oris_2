# HomeWork_Oris_2

## Описание
Данный проект представляет собой учебное веб-приложение для работы с турами.  
Проект состоит из клиентской части на React и серверной части на .NET API.

## Структура проекта
- `frontend` — пользовательский интерфейс на React
- `backend` — серверное API на .NET
- `README.md` — инструкция по запуску проекта

## Запуск frontend

```bash
cd frontend
npm install
npm run dev
npm run lint
```

## Запуск backend

```bash
cd backend
dotnet build
dotnet run
```
